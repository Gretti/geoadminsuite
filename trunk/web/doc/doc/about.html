What is GeoAdminSuite ?

GeoAdminSuite is a Web application to manage geographic data. It is composed of 4 logical parts :

Server config : a data source selector to choose databases and folders containing geographic data. Supported datasources type are, for the moment, ShapeFiles, PostGIS, Oracle Spatial, ECW, GeoTIFF and ERDAS Imagine images. The format collections is to increase soon.
Catalog: lists supported geographic data, grouped by data source and distinguished by type. Also displays informations about selected data (geodata quickview, data mining, number and type of attributes, number and type of objetcs, spatial reference system).
Composer: displays and handles geographic data selected from the catalog.
Data display can easily be managed using option menus. This part of the application intends to be a wizard for an easy mapping of your data.
Publisher: an easy way to publish the map prepared in the composer within a Mapfish based site, to export geographic data, or to download the Mapserver mapfile prevoiously ocnfigured.

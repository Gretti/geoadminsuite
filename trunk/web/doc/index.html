What is GeoAdminSuite ?

GeoAdminSuite1.0 is an application developped using Java technologies (Non-distributed Java EE architecture). The purpose is to administrate geographic data coming from heterogeneous sources (local files, databases located on different computers).
The application is supported by a standard Web browser. It is free software, licensed under the GNU General Public License (GPL) version 3. This means you are free to use it and to modify it, but if you distribute your modifications you must distribute the modified source code as well.

GeoAdminSuite was created and developped in 2003 by by Gretti N'guessan and Nicolas Ribot and is now maintained and developped by Camptocamp as an opensource project.

We hope you find this application useful for ... well, managing your data and maps, publishing websites or whatever it is able to do. You can find more of our work at camptocamp.

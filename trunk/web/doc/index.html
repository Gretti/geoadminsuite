What is GeoAdminSuite ?

GeoAdminSuite1.0 is an application developped using Java technologies (Non-distributed Java EE architecture). The purpose is to administrate geographic data coming from heterogeneous sources (local files, databases located on different computers or web services).
The application is supported by a standard Web browsers (actually it is really cool on FF only). It is free software, licensed under the GNU Lesser General Public License (LGPL) version 3. This means you are free to use it and to modify it.

GeoAdminSuite is an opensource project that was created and developped since 2003 by Gretti N'guessan and Nicolas Ribot. The team has recently grown up to include members of the Camptocamp team.

We hope you find this application useful for ... well, managing your data and maps, publishing websites or whatever it is able to do.

You can also help us, if you like, keeping on improving and hosting the application ...

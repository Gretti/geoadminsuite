GeoTools library

Download geotools 2.4.4 and remove the following jars, as they can cause problems: On Mac Os X: gt2-epsg-access-2.4.jar (failed to run on Apple Mac computers) On all plateforms: gt2-epsg-*-2.4.jar (KaboumServer can work with these libraries, but takes much longer to initialize datastores.
